# 📝 Notes Wall

A simple Notes Wall application built using HTML, CSS, and JavaScript.

## Features

- Add new notes
- Prevent empty notes
- Dynamic note rendering
- "No notes added yet" message
- Stores notes in a JavaScript array

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## What I Learned

- DOM Selection
- Event Listeners
- Arrays
- createElement()
- appendChild()
- removeChild()
- textContent
- Input Validation
